# marplux

## 0.1.0

### Minor Changes

- 5d58e3e: Initial release: `marplux agenda` inserts agenda recap slides before each
  heading section of a Marp deck. Supports stdin/stdout, `-o`, `--check`,
  `--heading-level`, `--title`, `--class`, and the `agenda-skip` /
  `agenda-overview` markers. Idempotent and zero runtime dependencies.
